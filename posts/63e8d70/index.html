手摸手教你拥有属于自己的博客

发表于2021-09-02|更新于2021-10-14|博客
|阅读量:

起因

      由于网上的教程过于久远，所以决定自我总结一下如何搭建，但并不是他们的教程不好。而是，牵扯到Github的更新，导致按照以前的配置去修改是会出错。

​ 下面教程就开始了，让我们开冲！

1、技术栈

  • node.js
  • hexo
  • 这俩也不算技术支持，算工具吧…
    • git
    • github

开始安装这些东西

2、安装与配置

2.1 node.js

描述。

2.1.1 安装

  • 下载地址：https://nodejs.org/en/
  • 选择LTS版本
  • 尽量安装到系统盘之外的地方，例如：E:\node_js

2.1.2 配置

配置一下环境变量即可。

  • 按win键，直接输入：编辑系统环境变量
  • 点开它
  • 在弹出的窗口中点击环境变量
  • 在系统变量下找到path，双击打开
  • 点击右侧的新建，输入你安装node.js的路径，下面是我的路径
  • E:\node_js\

2.1.3 测试

测试是否安装成功。

  • 打开cmd
  • 分别输入：
    • npm -v
    • node -v
  • 分别显示出版本号就是安装无误

2.1.4 安装一下淘宝镜像源

因为是国外镜像所以下载速度慢，用国内的就快咯

  • 打开cmd
  • 输入:
    • npm install -g cnpm --registry=https://registry.npm.taobao.org
  • 输入：
    • cnpm -v
  • 若显示版本号即安装成功

2.2 hexo

描述。

2.2.1 安装

  • 在系统盘以外的地方建立一个文件夹，并在这个文件夹下运行cmd
  • 输入：
    • cnpm install hexo-cli
  • 输入：
    • hexo -v
  • 若显示版本号即安装成功

2.2.2 配置

还是配置环境变量。

不过要注意：

  • 在环境变量的设置中，一定要将定位定到**.bin**目录下，例如E:\hexo\node_modules\.bin
  • 若路径跟我一致并复制了我的这个地址，一定要注意.bin前面可能会出现两个,只需要留一个

输入：

  • hexo -v
  • 显示版本号即安装成功

2.3 git

描述。

2.3.1 安装

  • 在系统盘之外的地方，新建一个git目录
  • 下载地址：https://git-scm.com
  • 在地址页面中点击downloads
  • 选择支持你自己操作系统的版本
  • 安装步骤：
    • 选择好安装路径之后
    • 一路点下一步即可。

2.3.2 配置

1、先配置环境变量。

  • 将路径定位到cmd目录下就可以
  • 打开cmd

2、输入：

  • git -v

  • 显示出命令帮助选项即可

3、在安装git的目录下找到git-bash.exe，打开

  • 输入：
    • git config --global user.name "用户名"
    • git config --global user.email "邮箱地址"
    • 引号里的用户名和邮箱地址要和GitHub的用户名和邮箱一样

2.4 github

描述。

2.4.1 注册

  • GitHub网址：https://www.github.com
  • 点击左上sign in 根据提示进行注册

注意：

​ 一定要记住注册的邮箱账号、设置的名称、登录密码

2.4.2 配置

1、新建仓库

  • 选择 New repository

  • 在输入框中输入仓库名，格式必须是用户名.github.io

    • 由于我已建过这个仓库，所以才会爆红

  • 勾选Add a README.md

  • 单击Create repository，完成

2、配置

3、正式搭建

3.1 建立源文件夹

先要在本地构建，然后后续用git提交到github上。

  • 在你的系统盘符之外的地方建一个名为Blog的文件夹（什么名字都可以，但为了见名知义就起个Blog）
  • 切换至Blog文件夹，并在此目录下运行cmd
  • 输入：
    • hexo init
  • 等待初始化完成

初始化完成，你的Blog文件夹下就会有很多东西，若没有，建议用管理员打开cmd再执行 hexo init命令

3.2 启动博客

  • 在Blog文件夹下，打开cmd
  • 输入：
    • hexo s
  • 根据提示的地址，打开浏览器
  • 输入网址：http://localhost:4000
